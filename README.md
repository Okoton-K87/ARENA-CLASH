# Arena Clash

**Arena Clash** is a platformer game built using Arcade Physics in Phaser. 

## Controls
- **A**: Move Left  
- **D**: Move Right  
- **Space**: Jump  
- **Mouse 1**: Aim  
- **Click Mouse 1**: Fire  
- **E**: Heal
